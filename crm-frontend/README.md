# Next-Gen CRM - Frontend

This is the client-side application for the Next-Gen CRM platform. It is built using React, Redux Toolkit, and Tailwind CSS.

## Features Implemented

- JWT authentication
- Persistent login
- Dashboard with charts
- Lead CRUD operations
- Activity timeline
- Responsive UI

## Tech Stack

React 18, React Router v6, Redux Toolkit, Axios, Tailwind CSS, Recharts

## Setup

1. Clone repo
2. npm install
3. npm run dev
